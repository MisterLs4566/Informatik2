Die Geschichte der Programmiersprachen

Die Anfangszeiten

Der erste richtige Computer wurde 1941 von Konrad Zuse mit Helmut Schreyer in Berlin entwickelt. Es handelte sich um einen Digitalrechner, den Zuse Z3, der die Größe eines Wandschranks hatte. Er gilt als erster funktionsfähiger Universalrechner der Welt. Er war schon programmierbar und verwendete schon das Binärsystem.
In den 1960er und 1970er Jahren wurden viele Programmiersprachen entwickelt, weil sich die ersten Heimcomputer etablierten. Eine wichtige Programmiersprache, die auf den späteren Heimcomputern oft verwendet wurde und bis heute noch eine große Rolle spielt war das 1964 entwickelte BASIC ("Beginner's All purpose Symbolic Instruction Code"). Die wichtigsten Computer in den 1970er-1990er Jahre verwendeten einen eigenen BASIC-Interpreter (Atari BASIC 1979, Commodore BASIC 1977...)

Die 1990er Jahre

1992 wurde die Sprache: "Hypertext Markup Language", kurz HTML veröffentlicht. Mit dieser "Auszeichnungssprache" wurde ein einfacher Austausch zwischen Dokumenten innerhalb des Internets möglich. Seitdem wurde das Internet immer größer und HTML verbreitete sich in einer sehr hohen Geschwindigkeit. Die neueste Version von HTML ist HTML5.2, die 2017 veröffentlicht wurde. Die kurz nach HTML veröffentlichten Sprachen: CSS und JavaScript ermöglichten einerseits viele Gestaltungsmöglichkeiten für Internetseiten und andererseits Logik für diese.

Heute

Dadurch, dass sich die Hardware heutiger Computer so stark verbessert hat, kann man heute auch in einer "langsamen" Interpretersprache programmieren, ohne den Geschwindigkeitsunterschied zu spüren. Die Menge an Programmiersprachen ist durch den immensen Fortschritt auch sehr stark angestiegen, was dazu geführt hat, dass man heute eine viel größere Auswahl an Sprachen hat. Zu den beliebtesten Programmiersprachen gehören JavaScript in Kombination mit HTML und CSS, Python und Java.
Die Programmiersprache Python wurde 1991 veröffentlicht. Sie wird heute für Datenverwaltung, aber auch für Lernzwecke eingesetzt.
