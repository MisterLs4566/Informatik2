Syntax

Auf dieser Seite wird die Syntax der verschiedenen Programmiersprachen gezeigt. Dafür schauen wir uns den Code einer einfachen "Hello World" Ausgabe an. in manchen Beispielen wird auch eine Variable oder eine if Abfrage verwendet, um Besonderheiten der Programmiersprache zu verdeutlichen.

BASIC

In Atari BASIC sieht ein einfacher Hello World Befehl so aus:
10 PRINT "Hello World"
Auffallend ist die Zahl am Anfang der Zeile. Diese ist beliebig auswählbar und zeigt dem Interpreter, in welcher Reihenfolge der Code ausgeführt werden soll. Die Nummerierung ist typisch für BASIC.
