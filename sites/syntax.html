Syntax

Auf dieser Seite wird die Syntax der verschiedenen Programmiersprachen gezeigt. Dafür schauen wir uns den Code einer einfachen "Hello World" Ausgabe an. in manchen Beispielen wird auch eine Variable, eine if Abfrage oder auch eine For-Schleife verwendet, um Besonderheiten der Programmiersprache zu verdeutlichen.

BASIC

In Atari BASIC sieht ein einfacher Hello World Befehl so aus:
10 PRINT "Hello World"
Auffallend ist die Zahl am Anfang der Zeile. Diese ist beliebig auswählbar und zeigt dem Interpreter, in welcher Reihenfolge der Code ausgeführt werden soll. Die Nummerierung ist typisch für BASIC.

Pascal

In Pascal sieht eine Hello World Ausgabe so aus:
program HelloWorld;
begin
 writeLn("Hello World");
end.
In Pascal muss man zuerst einen Programmnamen definieren. Der Code, der in der "begin" - Methode steht, wird vom Compiler zuerst ausgeführt. Am Ende des Programms muss man es mit der "end." - Methode abschließen. Das Semikolon am Ende trennt die Zeilen und Befehle voneinander.

C

in C sieht eine Hello World Ausgabe so aus:
#include <stdio.h>
int main()
{
 printf("Hello World");
}
In C muss um eine Konsolenausgabe zu erzeugen erst eine Bibliothek importiert werden. Eine Bibliothek ist nichts anderes, als Code, der bereitgestellt wird, um bestimmte Prozesse zu vereinfachen.stdio.h ist eine Standardbibliothek von C, die genau das ermöglicht. Mithilfe von "#include", können in C Bibliotheken importiert werden. Wie in Pascal muss auch in C zuerst eine Routine erzeugt werden, die vom Compiler gefunden und ausgeführt wird. Die main Funktion spiegelt in C immmer den Code wieder, der tatsächlich ausgeführt wird. printf ist der Befehl, um dann die Ausgabe zu erzeugen.

C++

Das ist die Hello World Ausgabe in C++:
#include <iostream>
int main()
{
 std::cout <<"Hello World";
}
Das Verfahren ist das Gleiche wie in C, man muss auch hier eine Standardbibliothek importieren und eine main-Funktion definieren in welcher man dann den Befehl ausführen kann. Bei "std::" handelt es sich um einen sogenannten Namensraum (eng.: "namespace"), mit dem die Befehle der Bibliothek iostream ausgeführt werden.
Alternativ kann man die Ausgabe auch so erzeugen:
#include <iostream>
int main()
{
 using namespace std;
 cout <<"Hello World";
}
in diesem Fall wird definiert, dass das namespace direkt verwendet wird. Dadurch braucht man "std::" vor dem Befehl nicht, um diesen auszuführen.

Python

Um eine Hello World Ausgabe zu erzeugen braucht man in Python diesen Code:
print("Hello World")
In Python braucht man keine Bibliothek zu importieren, um eine Ausgabe zu erzeugen. Außerdem benötigt der Python Interpreter auch keine main Funktion, um den Code auszuführen. In Python werden zusätzlich auch keine Semikola verwendet, um einen Befehl abzuschließen.
Hier ist ein weiteres Beispiel für Python Code:
#Das ist ein Beispiel
wert1 = 10
wert2 = 20
if wert < wert2:
 print(wert1+wert2)
Anders als in anderen Sprachen wird hier ein Doppelpunkt und eine Einrückung verwendet um kennzuzeichnen, dass der Code Teil einer Routine ist.

Java

In Java kann man eine Hello World Ausgabe so erzeugen:
public class dateiname
{
 public static void main(String[] args)
 {
  System.out.println("Hello World");
 }
}
In Java muss zurerst eine Klasse definiert werden, die den gleichen Namen wie die Datei hat. Die main Methode wird vom Compiler ausgeführt. der println Befehl sorgt für eine Konsolenausgabe. Java hat eine "strenge" Syntax, um etwas zu definieren muss anders als in Python der Datentyp (ganze Zahl, Dezimalzahl, Zeichenkette) immer mit angegeben werden.
Hier ist ein weiteres Beispiel für Java Code:
public class dateiname
{
 public static void main(String[] args)
 {
  //Das ist ein Beispiel

  int[] liste = new int[100];

  for(int x=0; x<100; x++)
  {
   liste[x]=x;
  }

  for(int x=0; x<100; x++)
  {
   System.out.println(liste[x]);
  }
 }
}
Dieser Code erstellt ein Array (ähnlich wie eine Liste und kann mehrere Werte speichern) mit 100 möglichen Werten, die alle aus ganzen Zahlen bestehen müssen (dafür sorgen die Zahl 100 in den Klammern und der Ausdruck: "int"). Dann wird der Liste 99 Mal eine Zahl zugeordnet. Anschließend wird das Array dann ausgegeben.

JavaScript

In JavaScript sieht eine hello World Ausgabe so aus:
alert("Hello World");
Anders als bei anderen Sprachen wird hier die Ausgabe nicht über eine Konsole getätigt, sondern es öffnet sich (wenn man den JavaScript Code mit Html verknüpft) ein Ausgabefenster, in dem der Text steht.
Eine weitere Möglichkeit, Text auszugeben sieht in JavaScript so aus:
document.write("Hello World");
In diesem Fall öffnet sich kein Fenster, sondern der Text wird direkt auf die Html Seite geschrieben.

PHP

In PHP kann man eine einfache Ausgabe so ausführen:
<?php
 print "Hello World";
?>
Dadurch, dass PHP Html Code an den Browser weitergibt, ist das Ergebnis gleich wie bei document.write in JavaScript. Man käme auch zum gleichen Ergebnis, wenn man in ein Html Dokument "Hello World" hineinschreiben würde.
